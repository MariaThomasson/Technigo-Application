Breakpoints

A breakpoint is the “point” at which a website’s content and design will adapt in a certain way in order to provide the best possible user experience. Breakpoints are pixel values that a developer/designer can define in CSS. When a responsive website reaches those pixel values, a transformation occurs so that the website offers an optimal user experience.
